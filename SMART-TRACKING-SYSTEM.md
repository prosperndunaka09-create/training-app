## 🎯 **HOW IT WORKS:**

### **Step 1: User Registration**
1. **User registers** → Gets unique referral code (e.g., "OPT-ABC123")
2. **Telegram notification** → Admin gets user email + referral code
3. **Account created** → Tasks locked until training

### **Step 2: Admin Creates Training Account**
1. **Go to Admin Panel** → Training tab
2. **Fill in ALL fields:**
   - Training Email: `training@example.com`
   - Assigned To: `User Name`
   - **User Email**: `user@example.com` (registered user)
   - **User Referral Code**: `OPT-ABC123` (from registration)
3. **Create Account** → Training account linked to user

### **Step 3: User Training**
1. **User gets credentials** via Telegram
2. **Login with training account** → Works perfectly!
3. **Telegram notification** → Admin notified of login
4. **Complete training** → Unlock personal account
